### [Task 1] Pickle Rick

#1	What is the first ingredient Rick needs? : `mr. meeseek hair`

#2	Whats the second ingredient Rick needs? : `1 jerry tear`

#3	Whats the final ingredient Rick needs? : `fleeb juice`
